# CIELA 🌍

CIELA is an interactive weather application (frontend) developed using HTML, CSS and JavaScript.

## Features

- Live weather information
- 5-hour forecast
- Dynamic weather themes
- Weather Journal
- Sun Cycle
- Glassmorphism UI
- Animated Earth background
- Hidden Easter Egg worlds (Hogwarts, Heaven, Hell, Area 51 and Space)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API
- Local Storage
